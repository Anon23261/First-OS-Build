# Quick Start Guide

1. Build the OS:
   ```bash
   ./tools/build.sh
   ```

2. Run in QEMU:
   ```bash
   ./tools/run.sh
   ```

3. Try the editor:
   - Press Ctrl+O to open a file
   - Press Ctrl+S to save
   - Press Ctrl+Q to quit

4. Example commands:
   - Open example: `open examples/hello_world/hello.txt`
   - Save file: Press Ctrl+S
   - Exit editor: Press Ctrl+Q
